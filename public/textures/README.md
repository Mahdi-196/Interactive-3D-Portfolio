# Globe Textures

Place your globe texture image here.

## Recommended Image:

**Filename:** `globe-map.jpg` or `globe-map.png`

**Type:** Equirectangular world map (2:1 aspect ratio)

**Recommended Size:** 2048x1024 or 4096x2048 pixels

**Free Resources:**
- https://www.solarsystemscope.com/textures/ (Free planet textures)
- https://flaticon.com (Search "world map equirectangular")
- NASA Visible Earth: https://visibleearth.nasa.gov/

## For Vintage/Sepia Look:
Use a sepia-toned or antique-style world map for the 1930s detective theme.

## How to Use:
1. Download an equirectangular world map image
2. Save it as `globe-map.jpg` in this folder
3. The globe will automatically load the texture
